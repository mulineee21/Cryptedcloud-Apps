# Cryptedcloud Apps
Aplicacion oficial de Cryptedcloud para acceder a nuestro servicio privado en la red TOR



**SUS FUNCIONES
  - Instala un cliente Tor en el equipo
  - Instala todas las dependencias faltantes
  - Crea un icono de applicacion (CryptedCloud) para acceder de manera más facil al servicio se vera como una aplicacion normal


IMPORTANTE: NUESTRA APLICACION INSTALA EL NAVEGADOR TOR EN EL SISTEMA, ES LA HERRAMINETA ESENCIAL PARA QUE PUEDA ACCEDER A NUESTRO SERVICIO, SI LO INTENTA CON UN NAVEGADOR NORMAL NO FUNCIONARA, NECESITA UN ENRUTADOR HACIA LA RED TOR, SI LO DESINSTALA LA APPLICACIÓN DEJARA DE FUNCIONAR.

Estan en versiones como la de Linux, Windows, Android

(Proximamente se lanzaran mas novedades para macOS y iOS/iPadOS)


Para más información visite https://cryptedcloud.netlify.app
